# Merkert Social Media

Online-Shop für personalisierte Social-Media-Posts für Vermögensberater.

## Schnellstart
1. `index.html` im Browser öffnen → Seite ansehen
2. `claude` im Terminal ausführen → Claude Code starten
3. Netlify Drop: netlify.com/drop → index.html hochladen → online

## Ordnerstruktur
```
merkert-social-media/
├── index.html          ← Die komplette Homepage
├── CLAUDE.md           ← Kontext für Claude Code (automatisch gelesen)
├── README.md           ← Diese Datei
├── fotos/              ← Annas Fotos hier ablegen
├── assets/             ← CSS/JS wenn später ausgelagert
└── posts/              ← Post-Texte und Inhalte
```
